## Basic Embedded Calendar

A simple embedded calendar with default settings.

**Full Configuration:**
```javascript
{
  displayMode: 'embedded',
  visibleMonths: 2,
  monthWidth: 500,
  selectionMode: 'range',
  position: 'bottom-left',
  useDynamicPosition: true,
  showHeader: true,
  showFooter: true,
  showSubmitButton: false,
  showTooltips: true,
  showLayersNavigation: false,
  showDateInputs: true,
  closeOnClickAway: true,
  startWeekOnSunday: false,
  primaryColor: '#0366d6',
  successColor: '#28a745',
  warningColor: '#f6c23e',
  dangerColor: '#dc3545',
  timezone: 'UTC',
  dateRangeSeparator: ' - ',
  baseFontSize: '1rem'
}
```

## Calendar with Restrictions

Calendar with date selection restrictions applied.

**Full Configuration with Restriction Factory:**
```javascript
{
  displayMode: 'embedded',
  visibleMonths: 2,
  monthWidth: 500,
  selectionMode: 'range',
  // ... other settings ...
  
  restrictionConfigFactory: () => ({
    restrictions: [
      {
        type: 'boundary',
        enabled: true,
        minDate: '2025-07-01',
        maxDate: '2025-12-31'
      },
      {
        type: 'weekday',
        enabled: true,
        days: [0, 6]  // Block weekends
      }
    ]
  })
}
```
