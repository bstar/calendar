# Getting Started

Welcome to the CLA Calendar component! This guide will walk you through the basic usage and features.

## Installation

```bash
npm install cla-calendar
```

## Basic Usage

The calendar can be used in two main modes:

### Embedded Mode
The calendar is displayed inline within your page layout.

<Canvas>
  <Story of={GettingStartedStories.BasicCalendar} />
</Canvas>

### Popup Mode  
The calendar appears as a popup when clicking an input field.

<Canvas>
  <Story of={GettingStartedStories.PopupCalendar} />
</Canvas>

## Core Concepts

- **Settings**: Configure display options, colors, and behavior
- **Selection Modes**: Choose between single date or date range selection
- **Callbacks**: Handle date selection events
- **Customization**: Style the calendar to match your design
